## Installation

Add this line to your application's Gemfile:

    gem 'rack-metrics', github: 'EvolutionSoft/rack-metrics'

And then execute:

    $ bundle

## Usage

If you have a Rails application use the `rails g rack:metrics:install` command to generate the initializer.
